## 📌 Description

**RS TOOLS** est un outil multi-fonction tout-en-un développé en Python 3, conçu pour les professionnels de la cybersécurité, les chercheurs en OSINT et les passionnés de pentest.

Il regroupe des modules avancés avec une interface terminal moderne (Rich) et une architecture modulaire propre.

---

## ✨ Fonctionnalités

### **1. Discord Tools**
- Token Checker avancé (User & Bot) + détection 2FA
- Webhook Spammer + Embed
- Webhook Info & Delete
- Fake Token & Nitro Generator

### **2. OSINT Tools**
- IP Lookup (Geo, ASN, WHOIS)
- Username Tracker (12 plateformes)
- Email OSINT complet (MX, WHOIS, etc.)
- Phone Number Analyzer
- Domain OSINT

### **3. Network Tools**
- Port Scanner rapide + Banner Grabbing
- DNS Lookup complet (A, MX, NS, TXT, SOA...)
- Website Scraper avancé (emails, tech stack, headers)
- Ping Tool avec statistiques
- WiFi Scanner
- **SSL/TLS Certificate Analyzer**

### **4. System Tools**
- Informations système détaillées (CPU, RAM, Disque, MAC...)
- Liste des processus

### **5. Extra Tools**
- Générateur de mots de passe forts
- Générateur d'identité fictive
- Visionneuse de résultats & Logs
- Nettoyage automatique
- Éditeur de configuration
